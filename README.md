# 🔌 SkillFlow for VS Code

Browse, search, and install AI agent skills from [SkillFlow](https://skillflow.builders) directly in your editor.

## Features

- **Search Skills** — Find AI skills by name, category, or keyword
- **Browse Marketplace** — Explore curated skills with trust scores in a webview panel
- **One-Click Install** — Install skills directly from VS Code terminal
- **Activity Bar** — Quick access to trending and installed skills

## Commands

| Command | Description |
|---------|-------------|
| `SkillFlow: Search Skills` | Search the marketplace |
| `SkillFlow: Browse Marketplace` | Open marketplace panel |
| `SkillFlow: Install Skill` | Install a skill by ID |

## Installation

1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X)
3. Search for "SkillFlow"
4. Click Install

Or install from terminal:
```bash
code --install-extension skillflow.skillflow-vscode
```

## Requirements

- VS Code 1.85.0+
- Python 3.8+ (for skill installation)
